# 하이, 데이터 (HY-DATA)

한양대 ERICA 데이터 포털사이트

## 프로젝트 소개
### 배경📒

분산된 학교데이터를 수집하는 과정에서 어려움을 겪은 경험을 바탕으로 서비스를 기획하게 되었습니다.

### 핵심가치🗝️

학교 데이터를 효율적으로 수집하고 시각화하여 유저에게 직관적이고 유용한 정보를 제공합니다.

### 기대효과💡

하이데이터에서 제공된 정보를 기반으로 다양한 서비스를 만들 수 있는 환경을 제공합니다.
